# RuoYi Shop

原生微信小程序电商骨架，使用 `JavaScript + WXML + WXSS`。

## 结构

- `pages/home` 首页
- `pages/category` 分类
- `pages/cart` 购物车
- `pages/profile` 我的
- `pages/search` 搜索
- `pages/product/detail` 商品详情
- `pages/order/list` 订单列表
- `pages/order/confirm` 订单确认
- `pages/address/list` 地址管理
- `pages/login` 登录

## 说明

1. 用微信开发者工具直接导入 `RuoYi-wx-app`
2. 入口页是 `pages/home/index`
3. 后端地址在 `app.js` 的 `apiBaseUrl`
