## 二、项目配置

### 1. 配置 ClickHouse 从库连接

编辑文件：`ruoyi-admin/src/main/resources/application-druid.yml`

```yaml
spring:
  datasource:
    # 从库配置 (ClickHouse)
    slave:
      enabled: true
      driverClassName: ru.yandex.clickhouse.ClickHouseDriver
      url: jdbc:clickhouse://localhost:8120/your_database
      username: your_username
      password: your_password
```

**注意：** 
- 端口号通常是 8123(HTTP) 或 9000(Native)
- 如果使用 Docker，可能是映射的端口如 8120

### 2. 添加 ClickHouse 依赖（如果还没有）

编辑文件：`ruoyi-JnBusViz/pom.xml`

```xml
<dependencies>
    <!-- ClickHouse JDBC -->
    <dependency>
        <groupId>ru.yandex.clickhouse</groupId>
        <artifactId>clickhouse-jdbc</artifactId>
        <version>0.3.2</version>
    </dependency>
    
    <!-- Lombok -->
    <dependency>
        <groupId>org.projectlombok</groupId>
        <artifactId>lombok</artifactId>
        <scope>provided</scope>
    </dependency>
</dependencies>
```

## 五、验证测试结果

### 成功标志 ✅
- 所有接口返回 `code: 200`
- `data` 字段包含实际数据（非空数组或对象）
- 控制台无数据库连接错误

### 常见问题 ❌

#### 问题 1: 返回 404
```json
{
  "timestamp": "2025-03-08T10:30:00.000+00:00",
  "status": 404,
  "message": "Not Found"
}
```
**解决方案：**
- 确认项目已完全启动
- 检查 Controller 路径是否正确
- 查看端口是否为 8080

#### 问题 2: 返回空数据
```json
{
  "code": 200,
  "msg": "操作成功",
  "data": []
}
```
**解决方案：**
- 检查 ClickHouse 是否有数据
- 验证年份月份参数是否正确
- 查看数据库表名是否与实体类匹配

#### 问题 3: 数据库连接失败
```json
{
  "code": 500,
  "msg": "获取连接失败"
}
```
**解决方案：**
- 检查 ClickHouse 服务是否运行
- 验证 `application-druid.yml` 中的用户名密码
- 确认防火墙允许端口访问

## 六、前端集成（可选）

### 如果你要开发前端大屏

#### 步骤 1: 创建 Vue3 项目
```bash
npm create vite@latest bus-dashboard -- --template vue
cd bus-dashboard
npm install
```

#### 步骤 2: 安装依赖
```bash
npm install echarts axios pinia element-plus
```

#### 步骤 3: 配置 Axios
创建 `src/utils/request.js`:
```javascript
import axios from 'axios'

const request = axios.create({
  baseURL: 'http://localhost:8080',
  timeout: 10000
})

export default request
```

#### 步骤 4: 创建 API 文件
创建 `src/api/busViz.js`:
```javascript
import request from '@/utils/request'

export function getDashboardStats(year, month) {
  return request({
    url: '/bus/test/dashboard',
    method: 'get',
    params: { year, month }
  })
}

export function getGisMapData() {
  return request({
    url: '/bus/stops/map/all',
    method: 'get'
  })
}
```

#### 步骤 5: 创建大屏组件
创建 `src/views/Dashboard.vue`:
```vue
<template>
  <div class="dashboard">
    <h1>公交数据可视化大屏</h1>
    <!-- 添加你的图表组件 -->
  </div>
</template>

<script setup>
import { onMounted } from 'vue'
import { getDashboardStats } from '@/api/busViz'

onMounted(async () => {
  const res = await getDashboardStats(2025, 5)
  console.log('大屏数据:', res.data)
})
</script>
```

## 七、性能优化（可选）

### 添加 Redis 缓存

#### 步骤 1: 配置 Redis
编辑 `application.yml`:
```yaml
spring:
  redis:
    host: localhost
    port: 6379
    password: 
    database: 0
```

#### 步骤 2: 在 Service 中添加缓存
```java
@Service
public class FactStationFlowServiceImpl implements FactStationFlowService {
    
    @Autowired
    private RedisTemplate<String, Object> redisTemplate;
    
    @Override
    public Long getTotalFlow(Integer year, Integer month) {
        String key = "bus:flow:total:" + year + "-" + month;
        
        // 尝试从缓存获取
        Long cached = (Long) redisTemplate.opsForValue().get(key);
        if (cached != null) {
            return cached;
        }
        
        // 查询数据库
        Long result = factStationFlowMapper.getTotalFlow(year, month);
        
        // 存入缓存（30 分钟）
        redisTemplate.opsForValue().set(key, result, 30, TimeUnit.MINUTES);
        
        return result;
    }
}
```

### 常见错误及解决方案

#### 错误 1: ClickHouse 驱动找不到
```
ClassNotFoundException: ru.yandex.clickhouse.ClickHouseDriver
```
**解决：** 在 pom.xml 中添加 ClickHouse JDBC 依赖

#### 错误 2: 数据源切换失败
```
Cannot determine embedded database driver class
```
**解决：** 检查 `@DataSource(SLAVE)` 注解是否正确

#### 错误 3: MyBatis XML 找不到
```
BindingException: Invalid bound statement
```
**解决：** 
- 检查 XML 文件是否在 `resources/mapper` 目录
- 检查 namespace 是否正确
- 清理并重新编译项目
